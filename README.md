# 🎮 Hand Betting Game

A web-based hand betting game built with Angular and TypeScript, inspired by Mahjong-style tiles. Players predict whether the next hand will be higher or lower while managing dynamic tile values and aiming to achieve the highest score.

---

## 🚀 Overview

This project was developed as a technical assessment to demonstrate:
- Frontend architecture using Angular
- State management and game logic
- Clean component structure
- UI/UX design and responsiveness

The game features multiple tile types with dynamic behaviors and includes a full gameplay flow from landing page to summary screen.

---

## ✨ Features

- 🎯 **Start New Game**
- 📈 **Higher / Lower Betting System**
- 🀄 **Mahjong-Inspired Tiles**
  - Number Tiles
  - Dragon Tiles
  - Wind Tiles
- 🔄 **Dynamic Special Tile Values**
- 🧠 **Game Logic & State Management**
- 🗂 **Draw Pile & Discard Pile Tracking**
- ♻️ **Reshuffle Mechanism**
- 📜 **Hand History Tracking**
- 🏆 **Top 5 Leaderboard**
- 💀 **Game Over Summary Screen**
- 🔁 **Replay Functionality**

---

## 🛠️ Technologies Used

- **Angular (Standalone Components)**
- **TypeScript**
- **SCSS**
- **HTML5**

---

## 🎮 How to Play

1. Start a new game from the landing page  
2. View your current tile  
3. Predict whether the next tile will be **Higher** or **Lower**  
4. Continue building your score  
5. Game ends when you make an incorrect prediction  
6. View results on the summary screen
